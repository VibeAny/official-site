# VibeAny Official Site

Static marketing site deployed on GitHub Pages using Astro.

## Local dev

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Configure GitHub Pages to deploy from the `gh-pages` branch via GitHub Actions (recommended) or use the `docs/` folder.

The `site` URL is set in `astro.config.mjs` for correct asset paths.
